2. Создать запрос, который вернет количество продуктов
в корзине для одного пользователя.

SELECT cart.user_id AS user_id, COUNT(cart.product_id) AS quantity_of_products FROM `cart` GROUP BY cart.user_id

3. Создать запрос, который посчитает общую стоимость
продуктов в корзине.

SELECT SUM(product.Price), cart.user_id AS user_id
FROM product 
INNER JOIN cart ON product.id = cart.product_id
GROUP BY  cart.user_id


4. Создать запрос, который посчитает среднее значение
стоимости продуктов в корзине.

SELECT AVG(product.Price), cart.user_id AS user_id
FROM product 
INNER JOIN cart ON product.id = cart.product_id
GROUP BY  cart.user_id
